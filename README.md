# Viewbot-V1.1
Viewbot on youtube

Slightly altered with different sleep times

Youbot is a Shell Script to increase views on Youtube using Tor ips.

Usage:
git clone 

cd youbot

service tor start

sudo ./youbot.sh URL

Install requirements (xdotool, proxychains, tor):

apt-get install -y xdotool tor proxychains
